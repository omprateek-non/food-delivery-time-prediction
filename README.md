##Car Selling Price Prediction: Decision Tree Regressor

What This Project Does
This project uses a Decision Tree model to predict how much a used car will sell for. We used the cardekho.csv dataset, 
where the main goal was to guess the 'Selling_Price'.

How I Did It
Got the Data Ready: Loaded car data, cleaned it up (like removing weird price entries), and turned text info into numbers the computer can understand.
Trained the Model: Taught a Decision Tree Regressor model using our prepared data.
Checked the Results: Our model was pretty good! After predicting prices, we found it achieved an R-squared of around 92.3%, meaning it explained a lot of the selling price variation. (Note: if you mentioned 81% earlier, please clarify which R-squared value you want to highlight, as the notebook output shows 92.3%).
